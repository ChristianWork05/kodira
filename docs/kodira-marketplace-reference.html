Nota para el agente: estas tres pantallas van dentro del AppShell existente (rail + topbar), usando los componentes de packages/ui. Distinguir los 3 tipos por badge (producto/paquete/servicio a medida). En la ficha, el botón es "Comprar ahora" para producto/paquete y "Solicitar presupuesto" para servicio a medida. En catálogo, el precio muestra el número para producto/paquete y "Presupuesto"/"Desde X" para servicio a medida. Conectar a los endpoints del Prompt 2 (GET /offerings con filtros, GET /offerings/:slug, GET /sellers/:id, GET /me/seller/offerings). 4 estados, animaciones del sistema, y que los filtros NO se desborden (wrap/scroll).
